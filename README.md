# Plataforma de Gestión de Residuos - Documentación

Esta es la documentación oficial de la Plataforma de Gestión de Residuos, una solución integral que conecta a generadores de residuos con gestores certificados para optimizar la recolección, tratamiento y disposición de residuos.

## 🚀 Inicio Rápido

### Instalación
```bash
npm install
```

### Desarrollo
```bash
npm start
```

### Construcción
```bash
npm run build
```
